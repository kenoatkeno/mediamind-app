# MediaMind

MediaMind is an autonomous AI-powered campaign planning and optimisation platform for SMBs, boutique brands, and micro media agencies in the UK.

## Operating Modes

1. **THE STRATEGIST**
   A conversational AI agent that builds optimum media mix plans from budget, audience, objective, and timing inputs. Executes campaigns for digital channels dynamically on user approval, and provides forecasting and readiness reports for broadcast channels (TV, Radio, Outdoor).

2. **THE WATCHDOG**
   An autonomous agent that monitors live digital campaigns via the Meta and Google Ads APIs. It reallocates budget within user-set guardrails hourly and alerts clients in plain English.

## Setup

1. Copy `.env.example` to `.env` and fill out the required variables.
2. The core application runs via a Vite + React frontend and an InsForge backend.
3. Node.js environment is required to operate the frontend development server.
